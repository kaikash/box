== README

Box is very useful app

You can:

* Post
* Delete
* Synchronize
* Sort
* e.t.c.

Also, you should create file <tt>.env</tt> in the root of the project. And it should contain:

DEV_SECRET_KEY_BASE=YOURSECRETKEY

TEST_SECRET_KEY_BASE=YOURSECRETKEY

PROD_SECRET_KEY=YOURSECRETKEY

FB_STORAGE_ID=YOURFBSTORAGEID

VK_STORAGE_ID=YOURVKSTORAGEID

FB_USER_ID=YOURFBUSERID

VK_ACCESS_TOKEN=YOURFBSTORAGEID

FB_ACCESS_TOKEN=YOURVKSTORAGEID
